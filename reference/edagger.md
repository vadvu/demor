# e-dagger

e-dagger

## Usage

``` r
edagger(age, mx, ...)
```

## Arguments

- age:

  Numeric array of age intervals

- mx:

  Numeric array with age specific mortality rates.

- ...:

  Optional. Additional arguments for
  [`LT()`](https://vadvu.github.io/demor/reference/LT.md) function.

## Value

Numeric array
