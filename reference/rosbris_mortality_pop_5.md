# Rosbris' Mortality and population (5-year interval)

Rosbris' Mortality and population (5-year interval)

## Usage

``` r
rosbris_mortality_pop_5
```

## Format

### A dataframe

- year:

  year

- code:

  number of Region

- territory:

  rural/urban/total

- sex:

  sex: both/male/femal

- age:

  age

- mx:

  ASMR

- N:

  Population

- Dx:

  Number of Deaths

- name:

  Region name
