# Rosbris' Fertility and population (1-year interval)

Rosbris' Fertility and population (1-year interval)

## Usage

``` r
rosbris_fertility_pop_1
```

## Format

### A dataframe

- year:

  year

- code:

  number of Region

- territory:

  rural/urban/total

- age:

  age

- fx:

  ASFR

- N:

  Population

- Bx:

  Number of Births

- name:

  Region name

## Source

Russian Fertility and Mortality Database. Center for Demographic
Research, Moscow (Russia). Available at
https://www.nes.ru/demogr-fermort-data
