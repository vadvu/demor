# Rosbris' Fertility and population (1-year interval)

Rosbris' Fertility and population (1-year interval)

## Usage

``` r
rosbris_fertility_pop_1
```

## Format

### A dataframe

- year:

  year

- code:

  number of Region

- territory:

  rural/urban/total

- age:

  age

- fx:

  ASFR

- N:

  Population

- Bx:

  Number of Births

- name:

  Region name

## Source

Rosbris [webpage](http://demogr.nes.ru/index.php/ru/demogr_indicat/data)
