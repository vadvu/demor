# Rosbris' Mortality and population (1-year interval)

Rosbris' Mortality and population (1-year interval)

## Usage

``` r
rosbris_mortality_pop_1
```

## Format

### A dataframe

- year:

  year

- code:

  number of Region

- territory:

  rural/urban/total

- sex:

  sex: both/male/femal

- age:

  age

- mx:

  ASMR

- N:

  Population

- Dx:

  Number of Deaths

- name:

  Region name

## Source

Russian Fertility and Mortality Database. Center for Demographic
Research, Moscow (Russia). Available at
https://www.nes.ru/demogr-fermort-data
