# Rosbris' Fertility and population (5-year interval)

Rosbris' Fertility and population (5-year interval)

## Usage

``` r
rosbris_fertility_pop_5
```

## Format

### A dataframe

- year:

  year

- code:

  number of Region

- territory:

  rural/urban/total

- age:

  age

- fx:

  ASFR

- fx1:

  ASFR, 1 order

- fx2:

  ASFR, 2 order

- fx3:

  ASFR, 3 order

- fx4:

  ASFR, 4 order

- fx5:

  ASFR, 5+ order

- N:

  Population

- Bx:

  Number of Births

- Bx1:

  Number of Births, 1 order

- Bx2:

  Number of Births, 2 order

- Bx3:

  Number of Births, 3 order

- Bx4:

  Number of Births, 4 order

- Bx5:

  Number of Births, 5+ order

- name:

  Region name

## Source

Rosbris [webpage](http://demogr.nes.ru/index.php/ru/demogr_indicat/data)
